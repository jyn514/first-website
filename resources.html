Security

A small list of easy and effective changes you can make today. If you have trouble with any of them, please contact me and I'll add detail or instructions.
These links are meant to be as accessible as possible. This is a topic I feel strongly about - everyone should have the right to digital privacy.

  • Use a password manager:
    Generate random passwords, store them securely, and access them easily with a single master password. You'll stop using the same password for every site or trying to remember which you used. I recommend KeePassX, but there are options like LastPass available as well.
  • Use https by default:
    When you send a password to a website, it's vulnerable while in transit. Others on the network, connected to the same wifi, or with access to the receiving server can see your passwords. The solution is encryption. The https:// protocol is a standard feature in the modern web, allowing behind-the-scenes encryption without requiring any configuration. Download the extension to have all your pages use encryption automatically.
  • Block trackers and persistant cookies:
    Advertisers today, including Google Ads, track your browser for 'analytics' whenever you visit their page. Prevent companies you don't know from tracking you with Privacy Badger, a no-configuration necessary blocker.
    Take note - this is not a traditional adblocker; it will allow ads if they don't set a tracker in your browser, and may block other tracking sites such as Facebook and Twitter if they follow you to other sites.

Privacy

If you're interested in further steps you can take to protect your privacy, the following take a bit more effort to set up or use, but are extremely effective.

  • PGP encryption through the GNUPG program allows all emails, texts, and files you send across the internet to only be accessible to those you choose. A picture encrypted with PGP could be uploaded to any server in the world without anyone being able to tell what it was.
  • Anonymous internet browsing using TOR Browser
  • Information on Internet privacy, hosted on github.com
